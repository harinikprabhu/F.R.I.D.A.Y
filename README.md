# F.R.I.D.A.Y
Personal Assistant made in Python

F.R.I.D.A.Y is a simple personal assistant written in Python. There are three modules in this program. 
  1. Calculator
  2. Monthly Temperature Predictor
  3. Country Database
 
Calculator:
  Calculator is a simple Python module that performs six simple calculations: Addition, Subtraction, Multiplication, Division, Modulus and Exponentiation. It also enables us to continue calculations with results from previous calculations.

Additional Features in Calculator:
  1. Detection of ZeroDivisionError
  2. Validating inputs for data type
